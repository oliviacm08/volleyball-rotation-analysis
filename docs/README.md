# Docs Directory

This folder contains the abbreviated and final versions of the research paper associated with the Volleyball Rotation Efficiency Analysis project.

---

## Contents

- **Volleyball_Rotation_Efficiency_Analysis_Draft.docx / .pdf**  
  Draft versions of the research paper. Contains:
  - Project objectives and motivation
  - Methodology including preprocessing, simulation, and analysis
  - Key results and figures
  - Discussion and conclusions

- **Volleyball_Rotation_Efficiency_Analysis_Final.pdf**  
  Final polished version of the research paper, suitable for submission as a research supplement.

---

## Notes

- Figures and tables referenced in the paper correspond to outputs in the `analysis/` folder.  
- The paper summarizes both **real and simulated analyses**, highlighting rotation efficiency, serve type distributions, and comparisons between real and simulated data.  
- Drafts are provided for transparency and to show the iterative development of the research.  
- Only the final version should be considered for submission purposes.
